FLEXBOX ROW

Use these three properties to create a Flexbox row layout. 


  .row {
  display: flex;
  flex-direction: row;
  flex-wrap: wrap;
  }
FLEXBOX COLUMN

Use this to create a Flexbox layout. 


  .column {
  display: flex;
  flex-direction: column;
  }

LINEAR GRADIENTS

This will create a background linear gradient from top to bottom. 


  .linear-gradient-background {
  background-image: linear-
  gradient (
  rgba(232, 102, 236, 0.3) 0%.
  rgba(232, 102, 236, 0.6) 100%.
  );
  }
BOX TRANSITION GLOW

Use transition and box shadows to glow on hover. 


  .code-card .code-header {
  border-radius: 8px;
  transition: all 0.5s ease-in-
  out;
  }
  .code-card:hover,
  .code-card:hover .card-header {
    box-shadow: inset 0px 0px 8px
    rgba(232, 102, 236, 1), 0 0 15px;
    rgba(232, 102, 236, 1);
    }
  }
OVERLAY CARD WITH TITLE

Use position properties and negativer margins to raise elements higher than their natural starting point.
